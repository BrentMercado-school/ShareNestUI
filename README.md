# ShareNestUI

ShareNestUI is a web-based community resource sharing platform where users can share items, browse available resources, and send borrow requests.

## Introduction
Many useful items are underused in communities. ShareNestUI was developed to make borrowing and lending easier, more organized, and more accessible through a simple web interface.

## Project Structure
This project is divided into two separate GitHub repositories:

- **Frontend Repository** – contains the user interface built with HTML, CSS, and JavaScript
- **Backend Repository** – contains the Django API and database logic

## Objectives
- Encourage community sharing
- Reduce unnecessary purchases
- Provide a system for borrowing and lending items
- Make item browsing and request management easy

## Features
- User authentication
- Dashboard for viewing all shared items
- Latest items section
- Browse items by category
- View item details in a modal
- Borrow request submission
- Add item functionality
- Profile management
- Borrow request tracking

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Django
- Database: SQLite

## Pages
- `dashboard.html`
- `browse-items.html`
- `my-items.html`
- `borrow-requests.html`
- `profile.html`

## How to Run
1. Clone the repository
2. Open the backend project
3. Run the Django server
4. Open the frontend files using Live Server or a local server
